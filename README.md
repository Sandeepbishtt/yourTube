
# yourTube

YouTube-clone using React Js/Redux/Material UI


## Screenshot

[![yourtube.png](https://i.postimg.cc/J7FRpkSP/yourtube.png)](https://postimg.cc/MXykTv6Q)

  
## Tech Stack

**Client:** React, Redux, Material UI



  
## Run Locally


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
